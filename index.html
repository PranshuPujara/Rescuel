Advanced setup

Want to build it
yourself?

Most people only need the APK above. This section is for custom setup and testing.

01
Get the project ready

Open the project on your computer and prepare the files needed for setup.

git clone git@github.com:aumsantoki99-web/Rescule.git && cd Rescuel && npm install
02
Add your service details

Connect your sign-in and map details so the app can work on your device.

03
Start a test preview

Use this step only if you want to test changes while building the app.

npx expo start
04
Create the Android app

Generate the installable Android app file for your phone.

cd android && .\gradlew clean && .\gradlew assembleRelease
APK file location
android/app/build/outputs/apk/release/app-release.apk
05
Install on Android Device

Move the APK to your Android device, allow Install Unknown Apps, tap the file, and sign in to start using Rescuel.
